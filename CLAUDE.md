# Rial Archive — Claude Guidelines

## Design Context

See [.impeccable.md](.impeccable.md) for the full design system. Summary of key principles:

### Aesthetic Direction
Clean and minimal (Linear/Vercel aesthetic). Data-forward, generous whitespace. The chart is the hero.

### Brand Personality
**Clear. Grounded. Honest.**

### Key Design Rules
1. **Green accent** (`#16a34a` light / `#22c55e` dark) — use sparingly on interactive elements and positive indicators only
2. **Zinc neutrals** over gray — warmer, less clinical (`zinc-50/100/200/500/900/950`)
3. **Card radius**: `rounded-xl` (12px)
4. **Typography**: Inter with `tabular-nums` for all financial values; `text-xs uppercase tracking-wider` for section labels
5. **Both themes equal** — every component must look polished in light AND dark mode
6. **Calm transitions** — 150–200ms ease, nothing that surprises the user
7. **No deep nesting** — one page background, one card surface, not deeper
